Bienvenido a la pagina Sara Soluciones. Este sistema permite administrar todas las áreas de la pagina según el rol de cada usuario. A continuación encontrarás una guía breve para aprovecharlo al máximo.

🔐 Inicio de Sesión y Recuperación de Contraseña

Ingresa con tu usuario y contraseña asignados. Si la olvidas, usa la opción “¿Olvidaste tu contraseña?” para recibir un correo con las instrucciones de recuperación.

🏠 Navegación General

En el sidebar izquierdo podrás encontrar los accesos a todos los módulos: Facturas, Producto, Categoria, Compra, Proveedor, Cliente, entre otros. Puedes cambiar entre ellos haciendo clic.

👥 Roles de Usuario

Cada usuario tiene permisos específicos según su rol:
- Administrador: Puede crear, editar, eliminar y ver todos los registros.
- Cliente: Puede ver y realizar compras según el área de capacitaciones y gustos personales.

⚙ Accesibilidad

El sistema incluye un módulo de accesibilidad con herramientas para adaptar la interfaz:

  • Modo claro / oscuro
  • Ajuste de tamaño de fuente
  • Resaltado de texto, enlaces y títulos
  • Alta o baja saturación, monocromo
  • Guía de lectura, cursor grande y detener animaciones

👨‍💻 Gestión de Clientes

Desde el módulo Clientes puedes consultar la lista del sistema, editar nombres del cliente o restablecer contraseñas según permisos.

💡 Recomendaciones Generales

  • Guarda tus cambios antes de salir del formulario.
  • No cierres el navegador mientras se esté enviando información.
  • Revisa tus permisos antes de intentar editar o eliminar registros.
